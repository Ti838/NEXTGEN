# Nextgen Youth Club - Membership Form 📝

![Next.js](https://img.shields.io/badge/next.js-%23000000.svg?style=for-the-badge&logo=next.js&logoColor=white) ![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB) ![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white) 

A highly sophisticated, professional, and dynamically generated web-to-pdf membership form designed exclusively for the **Nextgen Youth Club, Uttara (BD-0103)**. 

The application utilizes strict CSS injection and document framing architecture to guarantee **100% pixel-perfect print parity**. What you see on the screen is exactly what is generated in the A4 digital receipt layout, circumventing browser layout inconsistencies.

---

## ✨ Key Features

- **Pixel-Perfect A4 Print Generation:** Converts user input into an immaculate, fully structured A4 receipt using raw CSS payload injection into isolated popup `window` instances.
- **Strict Mandatory Validation:** Rigorous error handling ensuring all critical fields (Name, ID, Parents, DoB, Photo, Address, Educational Info, Preferences) are fully filled prior to rendering the high-fidelity receipt.
- **Dynamic Watermarking & Auto-Dating:** Automatically applies a receipt generation timestamp and embeds dynamic watermarks and submission dates on the final PDF.
- **Mobile Responsive Architecture:** Horizontal pan-scrolling capabilities on the strict A4 view frame, allowing smartphone users to engage without distorting the receipt's native proportions.
- **Content Protection:** Global right-click context menu blocking to deter unauthorized structural copying or image scraping.
- **Developer Attribution:** Custom built-in responsive dev-footer honoring the creator.

## 🛠️ Tech Stack

- **Framework:** Next.js (Version 15.3.2)
- **Library:** React (Version 19)
- **Styling:** Vanilla CSS (Zero-dependency, handcrafted for strict layout compliance)
- **Typography:** Inter (Google Fonts)

## 🚀 Getting Started Locally

Running this project locally is extremely straightforward. 

First, install the dependencies if you haven't already:
```bash
npm install
```

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) (or whichever localized port it provides, e.g. 3001) in your browser to interact with the application.

## 🌍 Deployment Strategy (Vercel)

This application is structurally engineered for seamless deployment on **Vercel**.

1. Push the code to a GitHub repository.
2. Link the repository to your Vercel Dashboard.
3. Click "Deploy". 
*(No advanced Next.js routing configuration or SSR overrides needed).*
